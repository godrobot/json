# App Builder Entities Documentation

This folder contains documentation for the simplified input formats of App Builder entities (pages, overlays, dialogs, and menus). These formats define how AI systems and developers can provide simplified input that gets parsed into full App Builder structures.

## Builder Entities

- [Component](component.md) - Component building blocks (sections, text, buttons, etc.)
- [Page](page.md) - Main application pages
- [Overlay](overlay.md) - Modal overlays and sheets
- [Dialog](dialog.md) - Alert dialogs and confirmations
- [Menu](menu.md) - Contextual menus and navigation
- [Globals](globals.md) - Global components (future implementation)
