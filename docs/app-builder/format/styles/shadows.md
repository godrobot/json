# Shadow System Documentation

## Overview

The Makyo shadow system provides elevation and depth to elements through drop shadows.

---

## Shadow

**Syntax:** `<value>`

**Available values:**
- `none` - No shadow
- `sm` - Small shadow (subtle elevation)
- `lg` - Medium shadow (moderate elevation)
- `xl` - Large shadow (pronounced elevation)

**Examples:**
```typescript
shadow: "none"
shadow: "sm"
shadow: "lg"
shadow: "xl"
```
