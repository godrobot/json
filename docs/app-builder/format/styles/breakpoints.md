## List of breakpoints
The following breakpoints are available in Makyo App Builder:
- phone
- phone-landscape
- tablet
- tablet-landscape
- desktop
- tv

There equivalent in styling is as follows:
- ph
- pl
- ta
- tl
- de
- tv


| Breakpoint | Prefix | Device Type |
|------------|--------|-------------|
| Phone | `ph:` | Mobile portrait |
| Phone Landscape | `pl:` | Mobile landscape |
| Tablet | `ta:` | Tablet portrait |
| Tablet Landscape | `tl:` | Tablet landscape |
| Desktop | `de:` | Desktop screens |
| TV | `tv:` | Large displays |
